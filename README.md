# LoL 적 스펠 타이머 오버레이 (Electron)

게임 중 **적 팀의 챔피언과 소환사 스펠을 자동으로 표시**하고,
스펠 아이콘을 **클릭하면 그 스펠의 쿨다운이 카운트다운**되는 오버레이입니다.
쿨 시작은 수동 클릭 방식(예: 적이 플래시 쓰는 걸 보면 클릭)이라 안티치트와 무관하게 안전합니다.

## 동작 원리

- Riot **Live Client Data API** (`https://127.0.0.1:2999/liveclientdata/allgamedata`)
  를 통해 실시간 게임 데이터를 가져옵니다. 게임 실행 중에만 열리는 로컬 API라 **API 키가 필요 없습니다.**
- 이 API는 self-signed 인증서를 쓰기 때문에 브라우저 `fetch`로는 막힙니다.
  그래서 Electron **main 프로세스**에서 인증서 검증을 끄고(`rejectUnauthorized: false`) 호출한 뒤,
  IPC로 화면(렌더러)에 넘깁니다.
- 챔피언/스펠 ID는 로케일에 영향받지 않는 `rawChampionName` / `rawDescription` 필드에서 추출해
  한국어·영어 클라이언트 모두에서 이미지가 정확히 매칭됩니다.
- 스펠 기본 쿨다운은 **Data Dragon**(`summoner.json`)에서 가져와 패치마다 자동으로 맞습니다.

## A. 집 PC에서 그냥 실행해보기 (개발 모드)

1. [Node.js](https://nodejs.org) 설치 (LTS 권장)
2. 이 폴더에서 터미널을 열고:

   ```bash
   npm install
   npm start
   ```

3. League of Legends 게임에 입장하면 적 정보가 자동으로 표시됩니다.
   (게임 대기 중에는 "게임 대기 중…" 화면)

## B. 포터블 .exe 만들기 (PC방용, 설치 불필요한 단일 파일)

> ⚠️ **이 빌드는 반드시 윈도우 PC에서** 하세요. (집 PC 등)
> 한 번만 만들어두면, 그 exe는 Node가 없는 PC방 컴퓨터에서도 더블클릭만으로 실행됩니다.

집 윈도우 PC에서, 이 폴더에서 터미널을 열고:

```bash
npm install
npm run dist
```

빌드가 끝나면 `dist` 폴더 안에 **`LoL-Spell-Timer.exe`** 단일 파일이 생깁니다.
이 exe 하나만 USB나 구글 드라이브에 올려두면 됩니다.

### PC방에서 쓰는 법

1. USB 꽂거나 드라이브에서 `LoL-Spell-Timer.exe` 다운로드
2. 더블클릭 → 바로 실행 (설치·관리자 권한 불필요)
3. 게임 입장하면 적 정보 표시

참고:
- 서명되지 않은 exe라 윈도우/백신이 "알 수 없는 게시자" 경고를 띄울 수 있습니다. "추가 정보 → 실행"으로 진행하면 됩니다.
- PC방은 재부팅하면 복원되므로 갈 때마다 다시 복사해야 합니다. (USB면 몇 초)
- 첫 실행 시 ddragon에서 챔피언/스펠 이미지를 불러오므로 인터넷 연결이 필요합니다.

## 사용법

- **타이틀바를 드래그**해서 창을 원하는 위치로 옮기세요.
- 적이 스펠을 쓰는 걸 보면 해당 **스펠 아이콘을 클릭** → 쿨다운 카운트다운 시작.
- **다시 클릭**하면 타이머 취소(잘못 눌렀을 때).
- 0이 되면 자동으로 사라지고 아이콘이 다시 밝아집니다.

## 주의

- 게임은 **테두리 없음(Borderless)** 모드를 권장합니다. (완전 전체화면이면 오버레이가 가려질 수 있음)
- Riot API로는 적 스펠의 실제 쿨 상태를 알 수 없습니다(안티치트). 그래서 클릭 기반 수동 방식입니다.

## 확장 아이디어 (선택)

- **스펠 가속 보정**: 우주적 통찰(+18)·광휘의 장화 등으로 줄어든 쿨을 반영하려면,
  `renderer.js`의 `toggleTimer`에서 `cd`를 `cd / (1 + haste/100)`로 계산하도록 바꾸면 됩니다.
- **클릭 통과 오버레이**: 마우스를 게임으로 통과시키려면 `main.js`에서
  `win.setIgnoreMouseEvents(true, { forward: true })`를 단축키로 토글하도록 추가.
- **단축키로 스펠 시작**: `globalShortcut`으로 키보드 단축키 → 특정 적 슬롯 타이머 시작.
